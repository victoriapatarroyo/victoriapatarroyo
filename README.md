## 🙋‍♀️ Sobre mí

Soy Desarrolladora Fullstack con formación en Ingeniería de Sistemas, enfocada en crear soluciones web robustas y escalables. Trabajo con **Spring Boot**, **React** y **Node.js**, y utilizo **Java**, **JavaScript** y **Python** para desarrollar aplicaciones eficientes respaldadas por bases de datos SQL.

Participo en todo el ciclo de desarrollo de software, aplicando metodologías ágiles, buenas prácticas y una comunicación constante para transformar necesidades de negocio en productos digitales funcionales y de alto impacto.
